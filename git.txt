Comandos git:

git init -> inicia el proceso de git

git add . -> añade al stage los ficheros modificados en la carpeta que hemos hecho el init

git commit -m "Comentario" -> añadir comentarios

git status

git log



git remote add origin direccion_ssh -> Creamos el directorio remoto vinculado a la direccion ssh

origin: Nombre del repositorio remoto

git remote -> muestra los directorios remotos vinculados a dicha carpeta



git push -u origin main -> La primera vez , para subir los ficheros al remoto, main es la rama principal

git branch -> nos informa de la rama donde estamos

git branch nombre_rama -> crea una rama
git checout nombre_rama -> nos mueve a la rama
